第一次使用請先安裝憑證 / First-time user: Please install the certificate first

0) 下載憑證/Download Certificate： 下載 DTRICloud-service-client.crt

1) 依下圖步驟安裝 / Follow the steps below to install (Windows / Chrome):

2) 依下圖步驟安裝（Windows / Chrome）：開啟下載的憑證安裝 / Open the downloaded certificate file to start installation.

3) 依下圖步驟安裝（Windows / Chrome）：安裝憑證 / Click "Install Certificate"

4) 依下圖步驟安裝（Windows / Chrome）：存放位置 請選擇"本機" / Select "Local Machine" for the Store Location.

5) 依下圖步驟安裝（Windows / Chrome）：憑證存放區 "瀏覽" / Certificate Store - Click "Browse"

6) 依下圖步驟安裝（Windows / Chrome）：選擇 受信任的跟憑證授權單位 然後下一步 / Select "Trusted Root Certification Authorities" and click "Next"

7) 依下圖步驟安裝（Windows / Chrome）：完成憑證安裝 / Complete the certificate installation wizard

8) 依下圖步驟安裝（Windows / Chrome）：Ctrl+Alt+Delete 開啟工作管理員 然後結束瀏覽器App / Press Ctrl+Alt+Delete to open Task Manager, then end the browser process (task)

9) 安裝完成後，重新開啟瀏覽器再登入。/ After installation is complete, restart your browser and log in again.
